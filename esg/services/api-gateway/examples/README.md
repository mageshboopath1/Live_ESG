# API Gateway Examples

This directory contains example scripts demonstrating API Gateway features.

## Example Files

### Caching
- `demo_cache.py` - Demonstrates Redis caching functionality
  ```bash
  uv run python examples/demo_cache.py
  ```

## Requirements

Examples require:
- Redis running on localhost:6379
- Environment variables configured in `.env`

## Usage

Run examples with UV:

```bash
uv run python examples/<example_file>.py
```

See individual example files for specific usage instructions.
